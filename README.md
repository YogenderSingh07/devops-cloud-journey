# 🚀 DevOps & Cloud Computing Journey

Welcome to my DevOps and Cloud Computing learning repository! This project serves as a living portfolio of my technical growth, hands-on labs, configuration files, and practical scripts as I transition from core fundamentals to cloud infrastructure architecture. 

As a continuous learner, I am building this repository to document my understanding of system administration, networking, automation, and modern deployment pipelines.

### 🟩 Phase 1: Linux Fundamentals & System Administration
- [x] **Core CLI Navigation & File Management** (`ls`, `cd`, `mkdir`, `touch`, `grep`, `find`)

- [x] **Web Server Deployment & Service Control** (Starting/stopping Nginx and Apache via `systemctl`)
