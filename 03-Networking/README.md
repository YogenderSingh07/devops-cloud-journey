# 🌐 Module 03 — Networking Fundamentals

Networking is a fundamental skill for Linux, DevOps, Cloud Computing, and Cybersecurity. This module contains my notes and practical learning as I study the core networking concepts that every DevOps engineer should understand.

The goal is to build a strong foundation before moving on to Docker, Kubernetes, AWS, and other cloud technologies.

---

---

## 🎯 Learning Objectives

By the end of this module, I aim to understand:

* How computers communicate over a network
* The OSI and TCP/IP networking models
* The role of network devices such as hubs, switches, and routers
* The difference between IPv4 and IPv6
* Public and private IP addressing
* Basic subnetting and CIDR notation
* How DNS translates domain names into IP addresses
* Common network ports and firewall concepts
* The client-server communication model

---

## 💡 Why I'm Learning This

Networking is the backbone of modern infrastructure.

A solid understanding of networking helps when working with:

* Linux servers
* Web servers (Apache & Nginx)
* Docker containers
* Kubernetes clusters
* Cloud platforms like AWS, Azure, and Google Cloud
* CI/CD pipelines
* Infrastructure troubleshooting
